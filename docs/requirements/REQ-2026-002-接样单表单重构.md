## 1. 需求描述

### 用户原话

委托书编号改为委托书编号，委托日期改为委托日期，增加委托书登记号，委托书，工程名称，委托单位，建设单位，监理单位，施工单位，见证单位，取样地点，见证人，见证人电话，送检人，送检人电话，接样人表单输入，应该记录操作人员（即接样人），检测环境与主要设备新增与编辑接样单不维护。增加判定依据（多选，根据报告类别过滤），检测依据（多选，根据报告类别过滤），检测参数（多选，根据判定依据与检测依据过滤）

### 理解后的需求描述

#### 1.1 字段重命名

| 现有字段                  | 变更后     | 说明                                   |
| ------------------------- | ---------- | -------------------------------------- |
| receiptCode（委托书编号） | 委托书编号 | 表单 label 变更，字段 key 可保持或重构 |
| receivedDate（委托日期）  | 委托日期   | 表单 label 变更，字段 key 可保持或重构 |

#### 1.2 新增字段（共 15 个）

| 需求字段       | 类型 | 来源                                    | 说明                                  |
| -------------- | ---- | --------------------------------------- | ------------------------------------- |
| 委托书登记号   | 文本 | 表单输入                                | 新增字段                              |
| 委托书登记日期 | date | 表单输入                                | 新增字段（原「委托书」修正）          |
| 工程名称       | 文本 | 从合同（Contract.projectName）带出      | 只读显示                              |
| 委托单位       | 文本 | 从合同（Contract.clientUnit）带出       | 只读显示                              |
| 建设单位       | 文本 | 从合同（Contract.buildingUnit）带出     | 只读显示（REQ-2026-001）              |
| 监理单位       | 文本 | 从合同（Contract.supervisionUnit）带出  | 只读显示（REQ-2026-001）              |
| 施工单位       | 文本 | 从合同（Contract.constructionUnit）带出 | 只读显示（已存在）                    |
| 见证单位       | 文本 | 从合同（Contract.witnessUnit）带出      | 只读显示（已存在）                    |
| 取样地点       | 文本 | 表单输入                                | 新增字段                              |
| 见证人         | 文本 | 从合同（Contract.witness）带出          | 只读显示（已存在）                    |
| 见证人电话     | 文本 | 从合同（Contract.witnessPhone）带出     | 只读显示（已存在）                    |
| 送检人         | 文本 | 表单输入                                | 新增字段                              |
| 送检人电话     | 文本 | 表单输入                                | 新增字段                              |
| 接样人         | 文本 | 表单输入，记录操作人员                  | 当前 `receivedBy`，label 改为"接样人" |

#### 1.3 检测环境与主要设备「不维护」

- **含义**：新建/编辑接样单时，`检测环境` 和 `主要设备` 两个字段**完全隐藏**
- 数据由别的环节（样品录入阶段）维护

#### 1.4 新增多选字段（共 3 组）

| 字段     | 数据来源                      | 过滤条件                                                            | 用途             |
| -------- | ----------------------------- | ------------------------------------------------------------------- | ---------------- |
| 判定依据 | 检测标准（TestStandard）多选  | 根据报告类别过滤                                                    | 预填后续数据录入 |
| 检测依据 | 检测标准（TestStandard）多选  | 根据报告类别过滤                                                    | 预填后续数据录入 |
| 检测参数 | 检测参数（TestParameter）多选 | 根据判定依据 ∪ 检测依据的**并集**过滤（即任一标准关联的参数都显示） | 预填后续数据录入 |
